In Python, data can be grouped in cataegories called types.

| name | example |
| --- | --- |
| string | `"hello"` |
| list | `[1, 2, 3, 4]` |
| int | `1` `-10` `23` |
| float | `3.14`, `-2.5`, `1.0` |
| bool | `True` `False` |

An example of using these types of data:

```python
fave_food = "tempura" # string
my_age = 15 # int
```
